# 🥩 FreshBazaar

A full-stack e-commerce platform for local meat shops and grocery stores.

Built with:
- Django REST Framework (Backend)
- Next.js (Frontend)
- PostgreSQL (Database)

---

## ✨ Features

### 👤 Authentication
- User registration & login
- JWT-based authentication

### 🏪 Shop System
- Shop owners can create their own shop
- Manage shop details

### 📦 Product Management
- Add / update / delete products
- Categorized products

### 🛒 Cart System
- Add to cart
- Update quantity
- Remove items

### 📦 Order System
- Checkout flow
- Order creation
- Order items tracking
- Shop receives order

---

## 🏗️ Architecture

- Clean modular Django apps
- Service layer pattern (business logic separated)
- Scalable structure for real-world usage

---

## 🧠 Tech Stack

| Layer     | Tech |
|----------|------|
| Frontend | Next.js |
| Backend  | Django REST Framework |
| Database | PostgreSQL |
| Auth     | JWT |
